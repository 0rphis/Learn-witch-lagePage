# 🏋️‍♂️ EnergyFit - Landing Page

Uma landing page moderna, responsiva e de alta conversão desenvolvida para uma academia de alto rendimento. O projeto foca em uma experiência de usuário (UX) fluida, utilizando animações de rolagem, carrossel de depoimentos e um formulário de contato funcional.

🌍 **Acesse o projeto online:** [EnergyFit ao vivo](https://my-test-laginpage.netlify.app)

---

## 🚀 Funcionalidades

- **Design Responsivo:** Layout adaptável para dispositivos móveis, tablets e desktops utilizando CSS Grid e Flexbox.
- **Animações Fluidas:** Elementos surgem suavemente na tela conforme o usuário rola a página (AOS).
- **Prova Social Dinâmica:** Carrossel interativo de depoimentos de alunos, com suporte a toque para dispositivos móveis (Swiper).
- **Planos de Assinatura:** Exibição de pacotes com destaque visual para o plano de melhor custo-benefício.
- **Contato Funcional:** Formulário integrado para envio de e-mails reais diretamente do front-end.
- **Navegação Suave:** Links âncora no cabeçalho direcionando perfeitamente para as seções da mesma página.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **[React](https://reactjs.org/)** (com **[Vite](https://vitejs.dev/)** para build ultra-rápido)
- **CSS3** (Estilização pura com variáveis, Grid e Flexbox)
- **[Lucide React](https://lucide.dev/)** (Ícones modernos e leves)
- **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)** (Animações de entrada)
- **[Swiper](https://swiperjs.com/)** (Carrossel de depoimentos)
- **[EmailJS](https://www.emailjs.com/)** (Integração de envio de e-mails sem back-end)
